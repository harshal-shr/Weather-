# 🌦️ Weather App

A beautiful, real-time weather dashboard that shows current weather and 5-day forecast for any city in the world. Features dynamic backgrounds that change based on weather conditions!

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🔍 **City Search** | Search weather for any city worldwide |
| 📍 **GPS Location** | Auto-detect your current location |
| 🌡️ **Current Weather** | Temperature, humidity, wind speed, visibility |
| 📅 **5-Day Forecast** | Next 5 days weather prediction |
| 🎨 **Dynamic Background** | Changes based on weather condition |
| 🌌 **Glassmorphism UI** | Modern frosted glass design |
| 📱 **Responsive** | Works on mobile & desktop |
| ⌨️ **Keyboard Support** | Press Enter to search |

---

## 🎮 How to Use

1. **Search by City:**
   - Type city name in search box
   - Click 🔍 or press Enter

2. **Use GPS:**
   - Click 📍 button
   - Allow location access

3. **View Details:**
   - Current temperature
   - Feels like, humidity, wind, visibility
   - 5-day forecast below
